####使用方法（控制台输入）:

	1+5
	>> 6

	(1+5)*2
	>> 12

####语法：      
	1.词法分析:      
		START		: NUMBER     
					| OPERATOR      
					|        
					;        
		NUMBER		: '0'..'9'+ ;       
		OPERATOR	: '+'|'-'|'*'|'/' ;       

	2.语法分析:
		expression  :	term { ("+" | "-") term }
		term		:	factor { ("*" | "/") factor }
		factor		:	number | "(" expression ")"
